Защита персональных данных (ФЗ-152)

Обработка персональных данных осуществляется в соответствии с Федеральным законом № 152-ФЗ.

1. Правовые основания обработки

ООО «АлтайЦентр» осуществляет обработку персональных данных на основании согласия субъекта данных и законодательства Российской Федерации.

2. Цели обработки

Персональные данные используются для оформления туристических услуг, страхования, а также для связи с клиентами.

3. Права субъекта персональных данных

  • Получать информацию об обработке своих данных;
  • Требовать уточнения, блокирования или уничтожения данных;
  • Отозвать согласие на обработку в любой момент.

4. Меры по защите данных

Компания принимает все необходимые организационные и технические меры для предотвращения несанкционированного доступа к персональным данным.

5. Контакт для обращений
